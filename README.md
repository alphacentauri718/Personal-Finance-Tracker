# Personal Finance Tracker

A full-stack web application for tracking personal finances, including assets, expenses, and net worth over time.

🔗 Live Demo: https://personal-finance-tracker-1n5c.onrender.com/

## Features
- User authentication (signup/login/logout)
- Asset and expense tracking
- Real-time net worth calculation
- Historical net worth visualization

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: Jinja2 templates + HTML/CSS
- Database: PostgreSQL
- Deployment: Render

## Motivation
Built to deepen understanding of full-stack development, database design, and deploying production-ready applications.

## Future Improvements
- Loan and payment calculators
- Investment portfolio tracking
- Integration with financial APIs (e.g., Plaid)
